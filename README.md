This is a [Next.js](https://nextjs.org) web app bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

This web application lets you search for and access the chapter or verse you want by typing commands preceded by a “/” and including the chapter title or verse number after the command, depending on the command.

## Getting Started

Clone the repo:

```bash
git clone https://github.com/Sharifian1/Quran-Search.git
````

Install dependencies:

```bash
npm install
```

Run the server:

```bash
npm run dev
```
Open http://localhost:3000 with your browser to see the result.
